### Essential Shortcuts (All Roles)

| Shortcut | Action | Use Case |
|----------|--------|----------|
| `/` | Open Global Search | Find any claim instantly |
| `N` | Create Fresh Claim | Start new claim workflow |
| `G + I` | Jump to Claims Inbox | View all claims |
| `G + A` | Jump to Acknowledgments | Check rejects |
| `G + E` | Jump to ERA & Payments | Review payments |
| `G + D` | Jump to Denials | Resolve denials |
| `G + R` | Jump to Reports | View analytics |

### How to Use Multi-Key Shortcuts
1. Press `G` (you have 1 second)
2. Press the second key (`I`, `A`, `E`, `D`, or `R`)
3. Page navigates automatically

### Power User Tips
- Memorize `G + I` for instant Claims Inbox access
- Use `/` to search from anywhere without clicking
- `N` creates a fresh claim from any page
- `Esc` closes search modal

## üìã Common Tasks

### Search for a Claim
```
1. Press `/` (opens Global Search)
2. Enter search query (patient name, DOS, claim ID, etc.)
3. Use faceted filters:
   - Patient Name
   - Date of Service
   - Billed Amount
   - Account/Serial/Invoice Numbers
4. Click search result
   ‚Üí Opens claim workspace
```

### Fix a Rejected Claim
```
1. Go to Claims Inbox (G + I)
2. Click "Rejected‚ÄîClearinghouse" or "Rejected‚ÄîPayer" tab
3. Review inline reject code (AAA02, CO-16, etc.)
4. Click "Fix" button
   ‚Üí Opens workspace at failing section
5. Review AI Copilot suggestion
6. Apply fix and resubmit
```

### Process an ERA Exception
```
1. Go to ERAs & Payments (G + E)
2. Click "ERA Details" tab
3. Filter to "Exception" status
4. Review mismatch category badge
5. Click "Review"
   ‚Üí Opens workspace at ERA section
6. Select mismatch category
7. Apply AI-suggested action
8. Create refund if needed
```

### Resolve a Denial
```
1. Go to Denials Workbench (G + D)
2. Sort by days remaining (urgent first)
3. Click "Resolve" on denial
   ‚Üí Opens workspace at Denials step
4. Review denial code and reason
5. Check AI-suggested End Action
6. Select End Action from dropdown
7. Complete workflow (resubmit or bill patient)
```

### Generate Patient Statement
```
1. Go to Claims Inbox (G + I)
2. Click "Ready to Bill" tab
3. Review patient responsibility amounts
4. Select claim(s) to bill
5. Click "Generate Statements"
   ‚Üí Opens Patient Billing workflow
6. Review statement preview
7. Email or print statement
```

### View Your Performance
```
For Coders/Billers:
1. Check dashboard "Performance Metrics" panel
2. Review accuracy, productivity, time metrics
3. Click "My Performance" quick action
   ‚Üí Opens Reports at Personal Metrics

For Managers:
1. Check "Team Performance" cards
2. Click "View Full Reports"
   ‚Üí Opens Reports at Team Performance tab

For Executives:
1. Check Executive Summary KPIs
2. Click "Full Reports"
   ‚Üí Opens comprehensive analytics
```

### Use AI Copilot
```
1. Open any claim in Claim Workspace
2. See AI Copilot right rail
3. Review context-aware cards (up to 8)
4. Click card to expand
5. Review suggestion and confidence score
6. Click "Preview Changes" (if available)
   ‚Üí Shows diff of old vs new
7. Click "Apply" to accept suggestion
8. Confirm in modal
```

### Access Medical Records
```
1. Open claim in Claim Workspace
2. Navigate to "Charge & Demographics" step
3. Click "Open Medical Records" link
4. See high-security red banner
5. Enter password (demo: "demo")
6. Click "Unlock Medical Records"
7. View documents (PHI masked by default)
8. Click "Unmask PHI" if needed (elevated access)
9. Review access log for compliance
10. Click "Close & End Session"
```
